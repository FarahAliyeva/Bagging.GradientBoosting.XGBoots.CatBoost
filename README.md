#  Loan Approval Prediction

This repository features a machine learning project designed to predict loan approval status (`Loan_Status`) based on customer profiles and financial histories. It covers comprehensive missing data analysis, data pre-processing, and evaluation of advanced boosting algorithms.

###  Tech Stack & Libraries
* **Language:** Python
* **Data Manipulation & Viz:** Pandas, NumPy, Matplotlib, Seaborn, Missingno
* **Machine Learning:** Scikit-learn, XGBoost, CatBoost

###  Dataset Features
The model evaluates the following independent variables to predict **Loan_Status**:
* Gender & Marital Status (Married)
* Dependents & Education level
* Applicant & Co-applicant Income
* Loan Amount & Loan Amount Term
* Credit History
